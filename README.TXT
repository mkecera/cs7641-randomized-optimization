Link to the code and data: https://github.gatech.edu/mkecera3/cs7641-randomized-optimization
All used libraries are saved in env.yml
Steps to run the code:
1. Install Anaconda https://docs.anaconda.com/anaconda/install/
2. Go to the folder and create environment: conda env create -f env.yml
3. Activate the environment: conda activate ml
4. Run Jupyter Server: jupyter notebook
5. A web browser should open. Open the assignment2.ipynb and run all cells
